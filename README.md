## Installation

Create a virtual environment:
```bash
$ python3 -m venv env
```

Start your virtual environment by running:
```bash
$ source env/bin/activate
```
Installing packages with requirements
```bash
$ cd GreeterGuru && pip3 install -r requirements.txt
```
